Xtract IS 4.12.0

The ABAP subdirectory now contains the following transport requests:

Z_THEO_READ_TABLE.zip

Transport request for new table custom function module Z_THEO_READ_TABLE. This is the successor to Z_XTRACT_IS_TABLE_COMPRESSION. Please consider adjusting your table extractions to use Z_THEO_READ_TABLE instead of Z_XTRACT_IS_TABLE_COMPRESSION.

Z_THEO_READ_TABLE-before_740SP05.zip

Same as above, but for older systems. This version does not support sub-selects in WHERE-clauses.

thtrans.zip

Transport request for Report component custom function Z_XTRACT_IS_REMOTE_REPORT and Table Join component custom function Z_XTRACT_IS_TABLE_JOIN.
